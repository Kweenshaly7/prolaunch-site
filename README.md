# ProLaunch Careers - Starter Website

This is a Vite + React + Tailwind starter website built by Mary-Queen Uchechukwu for ProLaunch Careers.

## Quick start (VS Code)

1. Install dependencies:
   ```
   npm install
   ```
2. Run development server:
   ```
   npm run dev
   ```
3. Build for production:
   ```
   npm run build
   ```
4. Preview production build:
   ```
   npm run preview
   ```

## Notes
- Fonts are loaded from Google Fonts (Playfair Display and Lato).
- Tailwind is configured in `tailwind.config.cjs`.
- Edit copy in `src/pages` and layout in `src/components`.
